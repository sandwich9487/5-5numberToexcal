# 簡介
此程式碼作為一次python與excal互動練習，功能為在jupyter上渲染5*5的按鈕，並儲存該按鈕對應的值(紅色為0、綠色為1)，並填入excal做為一筆資料(需自行添加label)，可提供於網路模型訓練。

# save_to_excel
點選保存後，會自動將矩陣以行為優先，逐行輸入excal，形成1*25的向量
